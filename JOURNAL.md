# A.U.R.O.R.A. Project Journal

## Project Overview: A.T.L.A.S. / A.U.R.O.R.A.
**A.U.R.O.R.A.** (Autonomous Utility Robot for Obstacle Recognition and Analysis) is an autonomous rover designed to be controlled via AI, voice interaction, or human intervention [cite: 54]. It features ultrasonic sensors and a camera for intelligent navigation and allows for custom AI development through a local self-hosted web server [cite: 55, 56].

## Hardware Specifications (from BOM & Code)
* **Controller:** Raspberry Pi 5 (8 GB RAM) [cite: 116].
* **Chassis:** 4WD Four Wheel Drive Smart Robot Car Kit (Acrylic) [cite: 101, 103].
* **Motors:** TB6612FNG Dual DC Motor Driver [cite: 96, 37].
* **Sensors:** HC-SR04 Ultrasonic Distance Sensor and Raspberry Pi Camera Module 3 [cite: 104, 94].
* **Display:** SSD1306 OLED via I2C [cite: 43, 44].
* **Power:** Samsung INR18650-30Q Li-ion batteries and a dual battery holder with an on/off switch [cite: 73, 99, 100].
